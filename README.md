# Data Science Projects

This repository contains data science projects completed as part of my MS in Business Analytics & Data Science at New Jersey City University.

## Projects

### Diet Effectiveness Analysis
Exploratory data analysis on a diet study dataset. Includes data cleaning, feature engineering, group comparisons, and visualizations using Pandas and Seaborn.

### Titanic Survival Analysis & Financial Data API
Data cleaning and imputation on the Titanic dataset, plus live financial data retrieval using CurrencyLayer and yfinance APIs.

### Salary Prediction Regression
Built and compared linear and polynomial regression models (degrees 2–4) to predict salary from years of experience. Achieved R² of 0.90 using scikit-learn.

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn, scikit-learn, yfinance
